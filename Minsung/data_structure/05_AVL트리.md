# 05. AVL 트리

## 01. AVL 트리란

AVL 트리는 이진탐색 트리의 문제점을 보완하기 위해 등장하였습니다.

이진 탐색 트리는 탐색 시간이 `O(logN)` 이지만 한 쪽으로 치우쳐진 편향 트리가 될 경우, 탐색 시간이  `O(N)` 이 됩니다.

따라서 트리의 균형을 맞춰준 것을 AVL 트리라고 합니다.

균형을 맞춘다는 것은 좌우 높이 차가 1이하가 되도록 하는 것입니다.
